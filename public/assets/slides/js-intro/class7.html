Loops

  • for - loops through a block of code a number of times
  • while - loops through a block of code while a specified condition is true
  • do/while - also loops through a block of code while a specified condition is true
  • for/in - loops through the properties of an object

The For Loop


  for (statement 1; statement 2; statement 3) {
    // code block to be executed
  }
  • Statement 1 is executed (one time) before the execution of the code block.
  • Statement 2 defines the condition for executing the code block.
  • Statement 3 is executed (every time) after the code block has been executed.

The For Loop

The For Loop Example


  for (i = 0; i < 5; i++) {
    console.log(i)
  }

Looping in Reverse


  for (i = 10; i >= 0; i--) {
    console.log(i)
  }

Break

Terminates the loop immediately and passes control over the next statement after the loop


  for (var i = 1; i < 10; i++) {
      if (i % 3 == 0) {
          break;
      }
  }
  console.log(i); // 3

Switch Case

The switch statement is a flow-control statement that is similar to the if else statement.


  switch (expression) {
    case value_1:
        statement_1;
        break;
    case value_2:
        statement_2;
        break;
    case value_3:
        statement_3;
        break;
    default:
        default_statement;
}

Global scope

If a variable is declared outside all functions, it is said to be defined in the global scope.

Local scope

Variables that are usable only in a specific part of your code are considered to be in a local scope. These variables are also called local variables.

Recursive Function

  • A recursive function is a function that calls itself. And this technique is called recursion.

Example: Factorial


5! = 5 x 4 x 3 x 2 x 1 = 120

How can you write Factorial by using for loop?

Example: Factorial


function factorial(n) {
  var result = 1;
  for (var i = n; i > 1; i--) {
      result *= i;
  }
  return result;
}

How can you write Factorial by using for recursive?

Recursive Factorial


function factorial(n) {
    if (n <= 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

Hoisting

  • In the compilation phase, the JavaScript engine moves all the function declarations (and also variable declarations) to the top of its current context. We say those function declarations are hoisted.
  • In the execution phase, the JavaScript engine just executes the code.

JavaScript function hoisting


showMe(); // an hoisting example

    console.log('an hoisting example');
}
