What You Should See

  • Click button to open modal.
  • If a user clicks close button or anywhere except modal it should close.

How to do it

  • First commit should be html/css code
  • Second commit should be a js functions
  • Create a pull request when you are done

Extra Challenges

Hints

  It will easier if you create html/css as it's shown in the picture first.

          You need to use position fixed to create a modal.
        
          Background color of modal is tricky. You can't use opacity to whole modal, it will change the opacity of whole modal. You can create a new element just for that one.
        
          You need 2 functions to open and close
        
          Closing when you click outside of modal is tricky. You can check if clicked element is not a modal. (Remember how to check what is the clicked element in js)
        
          You have to animate and remove modal after it. In order to do something after sometime you can use setTimeout.
