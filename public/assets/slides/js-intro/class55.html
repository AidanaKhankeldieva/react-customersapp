JavaScript has primitive data types and object types

An Object

  • An object can be created with figure brackets {…} with an optional list of properties. A property is a “key: value” pair, where key is a string (also called a “property name”), and value can be anything.

Creating an object


  var user = new Object(); // "object constructor" syntax
  var user = {};  // "object literal" syntax - Most used one

Properties


  var user = {      // an object
  name: "Seytech",  // by key "name" store value "Seytech"
  age: 3            // by key "age" store value 3
};

Files...

The resulting user object can be imagined as a cabinet with two signed files labeled “name” and “age”.

Accessing Objects

  • We can add, remove and read files from it any time.
  • Property values are accessible using the dot notation:.

            // get property values of the object:
  console.log( user.name ); // Seytech
  console.log( user.age ); // 3

Adding property(key)


            user.isAdmin = true;

Deleting property(key)

To remove a property, we can use delete operator:


            delete user.age;

Multiword property

We can also use multiword property names, but then they must be quoted:


var user = {
  name: "John",
  age: 30,
  "likes birds": true  // multiword property name must be quoted
};
Square bracket notation

We can also use multiword property names, but then they must be quoted:


var user = {};
// set
user["likes birds"] = true;
// get
console.log(user["likes birds"]); // true
// delete
delete user["likes birds"];

The “for…in” loop


  var user = {
  name: "Kushtar",
  age: 30,
  isAdmin: true
};

for (var key in user) {
  // keys
  console.log( key );  // name, age, isAdmin
  // values for the keys
  console.log( user[key] ); // Maksat, 30, true
}

THIS

In a function definition, this refers to the "owner" of the function.


var person = {
  firstName: "Aiday",
  lastName : "Baigazieva",
  id       : 5566,
  fullName : function() {
    return this.firstName + " " + this.lastName;
  }
};

Copying by reference


var message = "Hello!";
var phrase = message;

We have two independent variables, each one is storing the string "Hello!".

Copying by reference


var user = {
  name: "Aibek"
};

The object is stored somewhere in memory. And the variable user has a “reference” to it.

Copying by reference


var user = { name: "Aizat" };

var admin = user; // copy the reference

Now we have two variables, each one with the reference to the same object.

Copying by reference


var user = { name: 'Jantai' };

var admin = user;

admin.name = 'Yusuf'; // changed by the "admin" reference

console.log(user.name); // 'Yusuf', changes are seen from the "user" reference

Comparison by reference


var a = {};
var b = a; // copy the reference
console.log( a == b ); // true, both variables reference the same object

var x = {};
var y = {}; // two independent objects

console.log( x == y ); // false

Create

An array can store elements of any type


// mix of values
var arr = [ 'Apple', { name: 'Guliza' }, true, function() { alert('hello'); } ];
