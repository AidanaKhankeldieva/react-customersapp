Counter

Create counter with these features:

  • Face: initially should be sad and when count is done it should smile change color.
  • Start: When "start" button gets clicked, start counting till a given number(input).
  • Pause: When "pause" button gets clicked, pause counting.
  • Stop: When "stop" button gets clicked, stop counting & set count number to 0.
  • Count: If given number is larger then it should increase otherwise decrease.
