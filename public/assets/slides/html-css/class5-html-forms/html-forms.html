Html Forms

What are Forms?

HTML Forms are required when you want to collect some data from the site visitor. For example registration information: name, email address, credit card, etc

Form element

The HTML <form> element represents a document section containing interactive controls for submitting information.

Quick Note

The HTML forms submittion is handled with some backend language and JavaScript. In this course we will learn HTML side of it.

Form Elements

Form contain elements like:

  • text type inputs
  • email type inputs
  • password type inputs
  • textarea - multiline inputs
  • drop-down menus
  • radio buttons
  • checkboxes
  • etc...

and they are used to take information from the user.

Input Attributes

Input elements might contain attributes like:

  • type
  • name
  • id
  • class
  • placeholder
  • value
  • etc...

and they are used to take information from the user.

Form submission

Form is a container element like a <section> or <footer> element, but specifically for containing forms; it also supports some specific attributes to configure the way the form behaves

The action attribute defines the location (URL) where the form's collected data should be sent when it is submitted.

The method attribute defines which HTTP method to send the data with (usually get or post).

Button Element

In order to sumbit the data we have to add button of type submit

Or input type of submit

  • <button type="submit">Send your information </button>
    • or
  • <input type="submit" value="Send your information">

Label Element

Text labels can serve as a purpose to both sighted and blind users.

It also links label with single input by using "for" attribute

Required Attribute

HTML5 introduces client-side form validation without the need for JavaScript. When you attempt to submit a form, your browser will validate all of the fields and return an error if any of the fields is incorrect.

Textarea

The input field for the message is a <textarea>; a multiline text field.

Checkboxes

The checkbox is shown as a square box that is ticked (checked) when activated.

Checkboxes are used to let a user select one or more options of a limited number of choices

Use label tag for best accessibility

Fieldset and Legend

The HTML <fieldset> element is used to group several controls as well as labels (<label>) within a web form.

Phone in Inputs

<input> elements of type tel are used to let the user enter and edit a telephone number. Because its not formatted automatically, we use patterns Ex: pattern="[0-9]{3}-[0-9]{3}-[0-9]{4}"

URL/website in Inputs

<input> elements of type url are used to let the user enter and edit a URL.

DateTime in Inputs

<input type="datetime">

This feature is obsolete. Although it may still work in some browsers, its use is discouraged since it could be removed at any time. Try to avoid using it.

Use <input type="datetime-local"> instead

It does not apply any validation to entered values.

File Upload in Forms

Input type of "file" can be used to upload files

Ex: <input type="file" accept="file_extension|audio/*|video/*|image/*|media_type">
